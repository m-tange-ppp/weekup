# 作成時日付指定

**完了日**: 2026-05-13
**カテゴリ**: 機能追加

## 背景・原因

週目標・日記・KPT の新規作成が「今週 / 今日」固定だったため、
過去日付や任意週への記録作成がしづらく、入力漏れの後追い記録に対応しづらかった。

## 対応方針

DDD の責務分離を維持しつつ、UI で選択した日付/週を既存の作成処理へ渡す形で実装した。
週計算は既存の `WeekService` を利用し、重複制約や整合性は既存リポジトリ/フックの制約を活用した。

| ファイル                      | 変更内容                                                                           |

| `components/WeekSelector.tsx` | 前週/翌週ボタンと DateTimePicker を組み合わせた週選択 UI を追加                    |
